# Business Analytics Project
**Loan Defaulter's Prediction using Statistical Analysis**

The data provided for this project is from the loans department of a bank. Based upon certain attributes, it is given if there was a default by the customer on the given loan or not. Hence it is a classification problem requiring to apply relevant techniques to classify customers as defaulters or otherwise.
